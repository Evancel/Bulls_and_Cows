Calculating sum with a custom class method

Hey there! This problem might be a bit unpredictable, but give it a go and let us know how you do!
Create a Calculator class with an instance method called calculate that takes two integers as parameters and returns their sum. In the main method, create an instance of the Calculator class, read two integers from the user, pass them to the calculate method, and print the result.
Sample Input:
5
3

Sample Output:
8


Sample Input:
10
20

Sample Output:
30



Memory limit: 256 MB
Time limit: 5 seconds
